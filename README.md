Brasil Encantado

Brasil Encantado é um site de turismo desenvolvido como trabalho de faculdade utilizando HTML e CSS. O projeto conta com 5 páginas (inicial, destinos, galeria, suporte e login), design responsivo, galeria de imagens, descrições de pratos típicos brasileiros e formulários de contato.
